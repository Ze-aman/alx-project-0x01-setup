# alx-project-0x01-setup

## Project Overview

This project demonstrates a **Next.js** application with multiple pages, dynamic content, and user interaction features. It includes:

- Project setup with **Next.js**, **Tailwind CSS**, and **TypeScript**  
- Navigation between pages  
- Display of posts and users 
- Dynamic creation of posts and users via **modals**  
## Features

1. **Navigation**  
   - Header navigation between Home, Posts, and Users pages  

2. **Posts Page**  
   - Displays a list of posts using `PostCard`  
   - Fetches posts 
   - Add new posts dynamically using `PostModal`  

3. **Users Page**  
   - Displays a list of users using `UserCard`  
   - Fetches users
   - Add new users dynamically using `UserModal`  

4. **Responsive Layout**  
   - Grid layouts adapt for mobile, tablet, and desktop screens
